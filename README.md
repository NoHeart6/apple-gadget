# Apple Gadget

Website katalog dan penjualan iPhone dengan tampilan premium, dibangun menggunakan Next.js, React, TypeScript, Tailwind CSS, dan Vinext/Vite.

## Fitur

- Katalog 30 varian iPhone dengan pencarian dan filter kategori
- Detail produk lengkap, responsif, dan dapat di-scroll
- Pilihan kapasitas, warna, kondisi, harga, dan simulasi cicilan
- Keranjang belanja dan perhitungan subtotal
- Alur checkout dan pilihan metode pembayaran
- Formulir konsultasi, Trade In, setup, garansi, serta purnajual
- Tampilan responsif untuk desktop dan perangkat seluler

## Menjalankan Proyek

Persyaratan utama: Node.js 22.13 atau lebih baru.

```bash
npm install
npm run dev
```

Kemudian buka alamat yang ditampilkan oleh terminal.

## Pemeriksaan Produksi

```bash
npm run lint
npm run build
```

## Teknologi

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Vinext dan Vite

## Catatan

Harga, stok, nomor pesanan, dan pembayaran pada proyek ini masih berupa simulasi antarmuka. Integrasi produksi memerlukan database, dashboard admin, layanan pembayaran, serta konfigurasi komunikasi toko.

Gambar produk menggunakan aset resmi Apple yang tersedia secara publik. Apple Gadget merupakan proyek toko independen dan tidak berafiliasi dengan Apple Inc.
